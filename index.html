How to Install

  1. Download the resource pack.
  2. Open Minecraft and go to "Options."
  3. Select "Resource Packs."
  4. Click "Open Resource Pack Folder."
  5. Drag and drop the downloaded .zip file into the folder.
  6. Back in Minecraft, select the pack and enjoy!
